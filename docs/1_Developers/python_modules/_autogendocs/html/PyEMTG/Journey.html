GetBoundaryConstraintOutputValueByName(stringIn, floatOrStringOut='string')[source]¶

Return float value of boundary constraint. Which boundary constraint is determined by looking for input string in self.BoundaryConstraintOutputs. The first time string is found, the value of that boundary constraint is returned as the type specified by floatOrStringOut. If string is never found, then None is returned

Parameters:

  • stringIn (string) – string to search for in the “left-hand sides” of self.BoundaryConstraintOutputs

  • floatOrStringOut (string, optional) – how to output the returned value: “string” or “float”
